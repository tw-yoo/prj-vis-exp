Example Question & Answer

This example demonstrates filtering by groups.

1. Question:
What is the difference between the highest percentage in the Men group and the lowest percentage in the Women group?
2. Answer:
0.622
3. Operations Used:
Filter Find Extremum Difference
4. Explanation (Step-by-step):
1. Filter data to get only Men group values 2. Find the maximum percentage in Men: 0.752 (60 years and over) 3. Filter data to get only Women group values 4. Find the minimum percentage in Women: 0.13 (18 to 39 years) 5. Calculate the difference: 0.752 - 0.13 = 0.622
