Example Question & Answer

This example shows filtering by line/series.

1. Question:
What is the difference between Playoff teams and All teams in the year when Playoff teams had their second highest average payroll?
2. Answer:
21
3. Operations Used:
Filter Sort Nth Retrieve Value Difference
4. Explanation (Step-by-step):
1. Filter the data to get only Playoff teams values 2. Sort Playoff teams values in descending order 3. Find the 2nd highest value using Nth operation - 2nd highest value for Playoff teams: 109.9 (occurs in 2009) 4. Retrieve All teams' value in the same year (2009) - All teams in 2009: 88.9 5. Calculate the difference: 109.9 - 88.9 = 21
